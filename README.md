# mobile-de-filter

Chrome extension for hiding and managing listings on mobile.de search result pages.

## Features

- Adds a Hide button to mobile.de listing cards
- Stores hidden listing IDs locally in Chrome
- Automatically hides previously hidden listings when search results are opened again

## Development setup

Open Chrome and go to:

```text
chrome://extensions
```

Enable **Developer mode**, click **Load unpacked**, and select this project folder.
